## hexo博客使用指南

### 文章管理
   1. hexo new [layout] <title>     (layout: post | page | draft)
   2. hexo publish [layout] <title>

   layout会自动匹配、使用scaffolds目录下的同名文件格式


### 发布管理
   1. hexo server -s -p 5000 -i 192.168.1.1
   2. hexo generate --watch | --deploy
   3. hexo deploy
